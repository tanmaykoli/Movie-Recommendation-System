# 🎬 Movie Recommendation System Using Collaborative Filtering

## 📌 Project Overview
This project implements a Movie Recommendation System using User-Based Collaborative Filtering.

The system recommends movies to users based on similarity between user rating patterns.

---

## 🚀 Features
- User-Item Matrix Creation
- Cosine Similarity Computation
- Personalized Movie Recommendations
- RMSE Evaluation Metric
- End-to-End Recommendation Pipeline

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📂 Dataset
MovieLens Latest Small Dataset  
- 100,000+ ratings  
- 600+ users  
- 9,000+ movies  

---

## 📊 Model Evaluation
The model was evaluated using RMSE.

**Final RMSE:** 3.22

---

## 🧠 How It Works
1. Load rating data
2. Create User-Item Matrix
3. Compute cosine similarity between users
4. Generate weighted movie recommendations
5. Evaluate predictions using RMSE
